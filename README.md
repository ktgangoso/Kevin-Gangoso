# Portfolio Application

This is a simple professional portfolio project built using HTML, CSS, JavaScript, and Tailwind CSS. The portfolio showcases various sections including an about section, projects, and a contact form, all enhanced with animations for a better user experience.

## Project Structure

```
portfolio-app
├── src
│   ├── index.html          # Main HTML file
│   ├── css
│   │   └── styles.css      # Custom styles
│   ├── js
│   │   └── main.js         # Main JavaScript logic
│   ├── components
│   │   ├── header.html     # Header component
│   │   ├── about.html      # About section
│   │   ├── projects.html    # Projects section
│   │   └── contact.html     # Contact section
│   └── assets
│       └── animations.js    # Animation functions
├── tailwind.config.js      # Tailwind CSS configuration
├── package.json            # npm configuration
└── README.md               # Project documentation
```

## Features

- **Responsive Design**: The portfolio is designed to be responsive and looks great on all devices.
- **Smooth Animations**: JavaScript animations enhance the user experience throughout the application.
- **Modular Components**: The use of HTML components allows for easy maintenance and updates.
- **Custom Styling**: Tailwind CSS is used for styling, with additional custom styles defined in `styles.css`.

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd portfolio-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Build the project (if necessary):
   ```
   npm run build
   ```

5. Open `src/index.html` in your browser to view the portfolio.
